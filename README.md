## 📖 项目介绍 (Introduction)

**小申交流站** 是我（一名本科在读学生）为了深入掌握全栈开发技术而发起的开源项目。

在这个项目中，我将尝试打通从 **后端架构设计** 到 **前端交互实现**，再到 **自动化部署** 的全流程。不仅仅是一个论坛，更是一个**“全栈技术的试验田”**。未来计划引入 **AI 助手** 功能，探索 LLM 在社区场景下的应用。

目前项目处于 **🚧 早期开发阶段 (Work in Process)**，欢迎 Star ⭐️ 关注或提交 Issue 交流！

## 🛠️ 技术栈 (Tech Stack)

本项目采用当下最流行的前后端分离架构，追求高性能与开发体验。

| 领域 | 核心技术 | 说明 |
| :--- | :--- | :--- |
| **Backend** | 🐍 **Python + FastAPI** | 高性能异步 Web 框架，自动生成文档 |
| **Database** | 🗄️ **PostgreSQL / MySQL** | 关系型数据库 (配合 SQLAlchemy/Tortoise ORM) |
| **Frontend** | ⚡ **Vue 3 + Vite** | 现代化的前端框架与构建工具 |
| **UI Framework** | 🎨 **Tailwind CSS / Element Plus** | 极速构建美观的界面 |
| **State** | 🍍 **Pinia** | 轻量级状态管理 |
| **DevOps** | 🐳 **Docker & Nginx** | 容器化部署与反向代理 |
| **Tools** | 🔧 **Cursor + Git** | AI 辅助编程与版本控制 |

## 🌟 核心功能 (Features)

- [x] **用户认证体系**：JWT 登录/注册、OAuth2 集成
- [ ] **社区互动**：帖子发布（Markdown 支持）、评论、点赞
- [ ] **实时通讯**：基于 WebSocket 的站内私信/通知
- [ ] **后台管理**：基于 RBAC 的权限控制与数据看板
- [ ] **AI 实验室**：集成 LLM API，实现智能摘要与问答助手 (In Plan 🤖)
- [ ] **全栈部署**：Docker Compose 一键拉起环境

## 🚀 快速开始 (Getting Started)

### 前置要求
- Python 3.10+
- Node.js 18+
- PostgreSQL / MySQL
- Redis (可选)
